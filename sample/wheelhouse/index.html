pyflamegpu Wheels

pyflamegpu wheels can be installed via pip using the --extra-index-url or -f, --find-links arguments, and one of the uri's provided.

Using --extra-index-url

Description Command
All builds of pyflamegpu. Defaults to latest CUDA with visualisation/python3 -m pip install --extra-index-url whl/ pyflamegpu
pyflamegpu built with CUDA 11.2 with visualisation/python3 -m pip install --extra-index-url whl/cuda112-vis/ pyflamegpu
pyflamegpu built with CUDA 11.2 without visualistion/python3 -m pip install --extra-index-url whl/cuda112/ pyflamegpu
pyflamegpu built with CUDA 11.0 with visualisation/python3 -m pip install --extra-index-url whl/cuda110-vis/ pyflamegpu
pyflamegpu built with CUDA 11.0 without visualistion/python3 -m pip install --extra-index-url whl/cuda110/ pyflamegpu

Using -f, --find-links

Description Command
All builds of pyflamegpu. Defaults to latest CUDA with visualisation/python3 -m pip install --find-links whl/pyflamegpu/ pyflamegpu
pyflamegpu built with CUDA 11.2 with visualisation/python3 -m pip install --find-links whl/cuda112-vis/pyflamegpu/ pyflamegpu
pyflamegpu built with CUDA 11.2 without visualistion/python3 -m pip install --find-links whl/cuda112/pyflamegpu/ pyflamegpu
pyflamegpu built with CUDA 11.0 with visualisation/python3 -m pip install --find-links whl/cuda110-vis/pyflamegpu/ pyflamegpu
pyflamegpu built with CUDA 11.0 without visualistion/python3 -m pip install --find-links whl/cuda110/pyflamegpu/ pyflamegpu
